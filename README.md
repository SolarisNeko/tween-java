# tween-java | 渐变计算系统

## Introduction

Game tween - calculate engine的 Java 实现.

渐变计算系统

# Easy Integration

## maven

```xml

<dependency>
    <groupId>com.neko233</groupId>
    <artifactId>tween-java</artifactId>
    <version>0.0.1</version>
</dependency>
```

## gradle

```kotlin
implementation("com.neko233:tween-java:0.0.1")

```
